# Focus on Today 🚀

**Focus on Today** is a premium, minimal, and professional goal-tracking web application designed to help you stay productive and hit your daily targets. Built with a focus on clean aesthetics and user experience, it ensures your goals are always front and center.

![Preview](./images/Sun.svg) <!-- You can replace this with a proper screenshot later -->

## ✨ Features

-   **💎 Premium Design**: Modern Glassmorphism aesthetic with smooth transitions and 'Outfit' typography.
-   **💾 Data Persistence**: Automatically saves your goals and progress to `localStorage`. Your data stays even after a page refresh.
-   **📊 Dynamic Progress**: Real-time progress bar tracking with motivational quotes that change as you complete goals.
-   **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
-   **⚠️ Smart Validation**: Prevents checking off goals until they are actually written, with clear error feedback.

## 🛠️ Built With

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom properties, Flexbox, and modern Glassmorphism effects.
-   **JavaScript (ES6+)**: DOM manipulation and `localStorage` integration.
-   **Google Fonts**: 'Outfit' for a high-end look.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your favorite browser.

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/your-username/focus-on-today.git
    ```
2.  **Open the file**:
    Double-click `index.html` or use an extension like "Live Server" in VS Code.

## 📝 How it Works

1.  **Set your goals**: Type your top 3 goals for the day in the input fields.
2.  **Track progress**: As you complete a goal, click the checkbox. The progress bar will update automatically.
3.  **Stay motivated**: Reach 3/3 to unlock the final completion message!
